## Universal Style Foundation

**Always prepend to any prompt below:**

```
isometric pixel-art meets watercolor illustration, 64×64 grid basis, top-down
45-degree cabinet projection, weathered natural materials (wood, stone, earth),
earth-tone palette with gold/orange/blue accents, no pure black (use soil-dark),
slight hand-drawn linework (AI assisted but character evident), 2–3 pixel stroke
weight, strong shadow on bottom edge (depth cue), no reflections or glossiness,
aged map illustration style, detailed texture but readable at small sizes,
--style raw --ar 1:1 --q 2
```

## PROMPT TUNING TIPS

1. **If output is too bright**: add "muted palette" or "desaturated colors"
2. **If output is too cartoony**: add "realistic textures" or "hand-painted style"
3. **If output lacks detail**: add "high detail texture" or "weathered appearance"
4. **If output is too large for 64×64**: add "centered in frame" or "icon composition"
5. **If output has unwanted background**: add "transparent/no background" or "white isolated"
6. **If colonist looks generic**: add "character-driven" or "memorable expression"
7. **If biome feels wrong**: add specific color names ("golden-tan sand", "deep forest green")
8. **If shadow is wrong direction**: add "shadow bottom-right" or "light from upper left"
