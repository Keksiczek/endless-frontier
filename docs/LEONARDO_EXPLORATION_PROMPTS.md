# Endless Frontier — Leonardo.ai Exploration Tiles & POI Prompts

**Flexible art direction for world discovery sites**

These prompts work with any illustration style — not locked to watercolor. Mix and match styles per biome or discovery type.

---

## 📍 EXPLORATION TILE STYLES

### Style Modifiers (pick one or combine)

- **Illustrated** — Clean hand-drawn, storybook feeling
- **Painterly** — Oil/acrylic texture, atmospheric
- **Detailed realism** — Naturalistic but stylized
- **Ink & wash** — Black line work with color wash
- **Muted palette** — Desaturated, moody
- **Aged parchment** — Old map aesthetic
- **Concept art** — Concept game art style

## 🏛️ RUINS & ARTIFACTS

### Ancient Stone Ruins

```
Top-down landscape view, aged illustration style, earth tones, warm natural light.

Crumbling stone structure foundation, weathered carved blocks scattered in overgrown grass,
ancient architectural fragments still standing, moss and vines reclaiming surfaces, worn path
leading through site, wildflowers and grass filling spaces between stones, sense of history
and abandonment, archaeological interest, isolated ancient place.

Style: Illustrated + Painterly
Mood: Ancient, Forgotten, Discoverable
```

## 🛠️ LEONARDO.AI SETTINGS

**For best results with these prompts:**

| Setting | Value | Notes |
|---|---|---|
| **Model** | Leonardo Diffusion XL or Vision | Try both, Vision often better for illustration |
| **Canvas Size** | 1024×1024 | Enough detail for discovery sites |
| **Guidance Scale** | 7–8 | Balance creativity with prompt adherence |
| **Style Preset** | None (use prompt text instead) | Let prompt direct style |
| **Generations** | 4 variants | Pick best, iterate if needed |
| **Negative** | "pure black, neon, digital, modern" | Keeps aged authentic feeling |

## 📋 GENERATION WORKFLOW

1. **Copy entire prompt** (from "Top-down landscape" through "Mood:")
2. **Paste into Leonardo.ai prompt box**
3. **Select Style** (from Style Modifiers or let Leonardo choose)
4. **Generate 4 variants**
5. **Pick best variant**
6. **Optional: Refine** with upscaling or variations
7. **Export as PNG** for asset library
